Summary of the antecedent

In my last article, I successfully used Function check_events() and Function update_screen() to simplifiy the program. In this blog, I’d like to make the aircraft to move.

Response buttons

Every time we press a button, we register an event in PyGame. Events are all fetched through the method pygame.event.get(), so in the function check_events(), we need to specify which types of events to check.Each keystroke is registered as a KEYDOWN event. When we detect a KEYBDOWN event, we check to see if the key that was pressed is specific.For example, if we press the right arrow key, we increase the rect.centerx value of the ship to move the ship to the right.

Allow constant movement

When we hold down the right arrow, we want the ship to keep moving to the right until the player releases it.We will have the game detect the Pygame.keyup event so that we know when the player releases the right arrow;We will then combine the KEYDOWN and KEYUP events with a flag called moving_right to achieve the continuous movement.(moving_right is a Boolean variable that is set to True when the right arrow key is held down)

In my next blog, I’d like to enable the ship to move both to the right and to the left.
